var a = 100;
console.log(a);  // 100
a = 'str';
console.log(a);  // 'str'

二、过渡

        

        

            
先抛个砖，jquery 大多都是用来做什么事情的呢？jquery 可以选择在文档中的某一些元素然后进行一些相关操作，比如可以获取某些元素的文本内容，或者是在这个文档中删除这个元素等等；jquery 还能够对这些元素进行事件的绑定，通过触发事件来执行相关的函数；jquery 甚至还封装了 ajax 请求，帮我们做了不同浏览器版本的兼容。

        

        

            
三、选择器、筛选

        

        

            
我们在用 jquery 的时候，最想知道的是能利用它做什么，怎么去利用它。在上一节中提到了我们可以利用 jquery 选择在文档中的某一些元素然后进行一些相关的操作。那么在这里，我们就需要如何利用 jquery 的选择器。

        

        

            
我们利用 jquery 想要去获取文档中的某个元素，可以通过 jquery 封装好的选择器去选中它。例如：

        

        


<!-- index.html -->
<div id="div1">my id is div1</div>
<input class="content" value="my class name is content" />
<div>I am div</div>

<!-- JS -->
<script>
var a = $('#div1');     // 获取了 id 为 div1 的元素
var b = $('.content');  // 获取了 class 为 content 的元素
var c = $('div');       // 获取了 2 个 div 元素

// 我们可以通过 console.log() 的方式在控制台打印输出内容
console.log(a.text());    // 'my id is div1'
console.log(b.val());     // 'my class name is content'
console.log(c.length());  // 2
</script>


        

        

            
有的同学可能会问，后面的.text()那些都是什么意思？其实，对照的 HTML 文档，都可以基本猜得出来是什么意思。.text()为 jquery 封装好的获取该元素内的文本信息；.val()为获取该元素的 value 属性的值；.length() 为返回选中的元素的个数。

        

        

            
四、属性

        

        

            
我们知道一个标签中，是包含有很多的属性的，看代码说话。

        

        


<div id="one" class="d-1" data-id="15484">
    this is <b>text</b>
</div>


        

        

            
获取这个 div 元素的文本信息：$('#one').text();    // 'this is text'

        

        

            
获取这个 div 元素的 HTML 值：$('.d-1').html();    // 'this is <b>text</b>'

        

        

            
获取这个 div 元素的 data-id 属性：$('#one').attr('data-id');    // '15484'

        

        

            
当然还能够进行设置：

        

        


$('#one').text('new text');    // div 的文本信息变成 'new text'
...


        

        

            
还有非常多的函数可以获取各式各样的属性，还包括隐藏的 css 样式属性也是能够获取到的，如果想要知道如何使用的话，请查看 jquery 的手册吧。
